# Архитектура Frontend сайта DVFU Rectorat

## Обзор проекта
Фронтенд для сайта Совета ректоров вузов Дальневосточного федерального округа, разработанный на React 19.2.0 с использованием React Router для навигации.

## Технологический стек
- **React**: 19.2.0 - основная библиотека для построения UI
- **React Router DOM**: 7.1.1 - маршрутизация и навигация между страницами
- **CSS**: Модульные CSS файлы для стилизации компонентов
- **Create React App**: Базовая настройка проекта

## Структура проекта

```
src/
```

## Компоненты

### Layout
Обертка для всех страниц, содержит Header, основной контент и Footer.
- Обеспечивает единообразие всех страниц
- Автоматически добавляет шапку и подвал к каждой странице

### Header
Синий хедер с навигацией и детальной картой России:
- **Левая часть:**
  - Ссылка на Российский союз ректоров (верхний левый угол)
  - Название "СОВЕТ РЕКТОРОВ ВУЗОВ ДАЛЬНЕВОСТОЧНОГО ФЕДЕРАЛЬНОГО ОКРУГА"
  - Краткое описание цели создания совета
  - Кнопка "Читать далее" → переход на страницу "О совете"
- **Правая часть:**
  - Горизонтальное меню навигации (расположено НАД картой)
  - Детальная SVG карта России с федеральными округами
  - Дальневосточный ФО выделен синим цветом с разбивкой по субъектам:
    - Республика Саха (Якутия)
    - Хабаровский край
    - Приморский край (особо выделен)
    - Амурская область
    - Камчатский край
    - Сахалинская область
    - Магаданская область
    - Чукотский АО
    - Еврейская АО
  - Интерактивность: подсказки при наведении на регионы
- Адаптивный дизайн для всех устройств

### Footer
Информационный подвал:
- Контактные данные
- Ссылки на полезные ресурсы
- Copyright информация

## Страницы

### Home (Главная)
- **Секция приветствия**: Фото и речь председателя совета
- **Секция новостей**: Карточки последних новостей (3 шт.)
- Все элементы интерактивны с hover эффектами

### About (О совете)
Вкладочная структура:
1. **Структура совета**: Диаграмма иерархии (Председатель → Заместители/Секретариат → Члены)
2. **Состав совета**: Таблица с регионами, университетами, ректорами и ссылками
3. **Региональные советы**: Таблица региональных советов ректоров

### Documents (Документы)
Категоризированный список документов:
- История создания
- Учредительные документы (положение, устав)
- Решения совета (протоколы заседаний)
- Кнопки скачивания для каждого документа

### Events (Мероприятия и новости)
- Фильтрация по типу: Все / Мероприятия / Новости
- Карточки событий с изображениями
- Бейджи для визуального различия типов
- Даты и описания событий

### Press (Для прессы)
5 вкладок:
1. **Пресс-релизы**: Информационные сообщения
2. **Анонсы**: Предстоящие мероприятия
3. **Фотоматериалы**: Галерея фото
4. **Видеоматериалы**: Видео с иконкой воспроизведения
5. **Контакты для СМИ**: Информация пресс-службы

### Contacts (Контакты)
- **Секретариат**: Полная контактная информация с иконками
- **Региональные советы**: Сетка карточек по регионам ДФО

## Роутинг

```javascript
/ → Home
/about → About
/documents → Documents
/events → Events
/press → Press
/contacts → Contacts
```

Все маршруты обернуты в Layout компонент для единообразия.

## Дизайн и стилизация

### Цветовая палитра
- **Основной синий**: `#1e4976` (темный)
- **Акцентный синий**: `#2B5A8E` (средний)
- **Светлый синий**: `#2d5f8d` (для градиентов)
- **Фон**: `#f8f9fa` (светло-серый)
- **Зеленый акцент**: `#4CAF50` (для новостей)

### Типографика
- Семейство шрифтов: System fonts (San Francisco, Segoe UI, Roboto)
- Заголовки: 600-700 weight
- Основной текст: 16px, line-height 1.6

### UI/UX особенности
- **Hover эффекты**: Поднятие карточек (translateY), изменение теней
- **Transitions**: Плавные переходы 0.3s
- **Box shadows**: Для карточек и элементов (2-16px размытие)
- **Border radius**: 6-8px для современного вида
- **Адаптивность**: Mobile-first подход с media queries

## Адаптивность

### Breakpoints
- **Desktop**: > 1200px - полная версия
- **Tablet**: 768px - 1200px - адаптивная сетка
- **Mobile**: < 768px - одноколоночный layout

### Адаптивные изменения
- Header: Вертикальная раскладка на планшетах
- Таблицы: Горизонтальный скролл на мобильных
- Сетки: Переход от multi-column к single-column
- Навигация: Вертикальное меню на мобильных
